โปรเจกต์นี้ประกอบด้วย 2 ส่วนการทำงานหลัก ได้แก่ ฝั่ง Server (Node.js) และฝั่ง AI API (Python)

## วิธีการรันเพื่อทดสอบบนเครื่อง (Local Development)

ทุกครั้งที่ต้องการเปิดทดสอบระบบบนเครื่อง Local ต้องเปิด Terminalขึ้นมา 2 หน้าต่างเพื่อรันทั้ง 2 ส่วนให้ทำงานพร้อมกันเสมอ

### 1. รันฝั่ง Python API (AI Model)

เปิด Terminal หน้าต่างที่ 1 (ตรวจสอบให้อยู่ในโฟลเดอร์ที่มีไฟล์ `api.py`)
แล้วรันคำสั่ง uvicorn api:app --reload --port 8000

### 2. รันฝั่ง Node.js Server

เปิด Terminal หน้าต่างที่ 2 (ตรวจสอบให้อยู่ในโฟลเดอร์ที่มีไฟล์ server.js)
แล้วรันคำสั่ง node server.js
